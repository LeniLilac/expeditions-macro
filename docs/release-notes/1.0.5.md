# Expeditions Macro 1.0.5

This patch makes Expedition prestart recognition independent of player avatars and tolerant of the dialog-size variation seen across Roblox installations.

## Changes

- Prestart recognition now locates the green Start Game control within a bounded center search area instead of comparing the entire fixed-size dialog.
- The detector confirms the stable dark header and neutral title text while completely excluding the dynamic player-avatar row.
- Button position and dimensions tolerate the observed 164 × 20 through 180 × 22 render sizes at the standard 808 × 611 Roblox client size.
- Regression coverage includes the newly reported avatar and dialog-scale variant plus every existing non-start UI capture to guard against false positives.
- Existing detector pack 1.0.1 installations receive the fix through the application update and do not need to recreate models or download a replacement pack.

## Downloads

- `ExpeditionsMacro-1.0.5-win-x64-setup.exe` — recommended Windows installer.

Windows 10 or Windows 11 x64 is required. The binaries are currently unsigned, so Windows may identify the publisher as unknown. Verify the checksums before running a downloaded artifact.
