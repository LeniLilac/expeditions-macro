# Expeditions Macro 1.0.13

This release makes camera-region setup independent of the current Roblox window size and adds a configurable global macro hotkey.

## Changes

- **Select region** now resizes Roblox to the standard 808 by 611 client before the overlay appears. The selected area therefore always fits the size used by camera calibration.
- Camera selections are converted to Roblox-client-relative coordinates immediately. Roblox is restored after a selection, a cancellation, or an error, and moving the window before setup no longer offsets the saved region.
- Added **Settings > Controls** with an inline hotkey recorder. F6 remains the default; F1-F11 and F13-F24 are supported, while F12 is rejected because Windows reserves it for debuggers.
- The selected key is saved across restarts and updates the Expeditions buttons, workflow instructions, diagnostic capture text, operation prompts, and sidebar start/stop hint.
- Added the app version to the bottom of the left navigation footer.
- Added regression coverage for standardized selection, relative coordinate conversion, preview capture, cancel/error restoration, hotkey defaults, rebinding, and reserved-key rejection.

Existing camera models, placement models, presets, and detector pack 1.0.1 remain compatible. The detector-pack version is unchanged because these changes affect window setup and application controls rather than UI-state detection.

Windows 10 or Windows 11 x64 is required. The binaries are currently unsigned, so Windows may identify the publisher as unknown. Verify the checksums before running a downloaded artifact.
