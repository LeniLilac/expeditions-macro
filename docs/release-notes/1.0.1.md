# Expeditions Macro 1.0.1

This patch improves camera-model setup and corrects two layout defects in the unified interface.

## Changes

- Camera-model setup now enables shift lock automatically before capturing and learning the yaw turn, then turns it off after success, cancellation, or failure.
- Standalone **Auto align** manages shift lock the same way. The full Expeditions workflow keeps its existing single shift-lock scope around pitch and yaw preparation.
- Camera-region coordinates no longer overlap **Select region** or **Show 30% overlay**.
- Text in single-line inputs remains vertically centered and fully visible, including while focused and at Windows display scaling.
- Developer UI snapshots can be rendered while the normal app instance is open; normal launches remain single-instance.

## Downloads

- `ExpeditionsMacro-1.0.1-win-x64-setup.exe` — recommended Windows installer.
- `ExpeditionsMacro-1.0.1-win-x64.zip` — portable self-contained build.
- `anime-expeditions-expeditions-1.0.0.zip` — unchanged compiled detector pack for update and rollback workflows.
- `SHA256SUMS.txt` — SHA-256 checksums for every release asset.

Windows 10 or Windows 11 x64 is required. The binaries are currently unsigned, so Windows may identify the publisher as unknown. Verify the checksums before running a downloaded artifact.
