# Expeditions Macro 1.0.0

The first release provides camera alignment, placement-model tooling, and unattended Expeditions workflows in one Windows desktop application.

## Highlights

- Start a configured Expeditions run directly from the Roblox lobby with F6.
- Learn and reuse full-turn camera models for large and fine yaw corrections.
- Record, edit, and test relative unit placements in one interface.
- Detect gameplay pauses, rewards of any rarity, checkpoints, victory, defeat, lobby returns, and Roblox disconnects.
- Extract at the first checkpoint or after a configurable number of boss nodes, while still handling early defeat.
- Send optional Discord Components V2 summaries with runtime, totals, recovery events, and screenshots.
- Use dark, light, or system theme in a unified, Linear-inspired native interface.

## Downloads

- `ExpeditionsMacro-1.0.0-win-x64-setup.exe` — recommended Windows installer.
- `ExpeditionsMacro-1.0.0-win-x64.zip` — portable self-contained build.
- `anime-expeditions-expeditions-1.0.0.zip` — compiled detector pack for update/rollback workflows.
- `SHA256SUMS.txt` — SHA-256 checksums for every release asset.

Windows 10 or Windows 11 x64 is required. The binaries are currently unsigned, so Windows may identify the publisher as unknown. Verify the checksums before running a downloaded artifact.
