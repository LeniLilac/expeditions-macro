# Expeditions Macro 1.1.1

This reliability release fixes the Challenge cooldown handoff and makes the public setup walkthrough available directly from the app.

## Fixed

- Challenge mode can now start the configured Expeditions preset while regular Challenges are unavailable without touching WPF controls from the background automation thread.
- Cooldown-handoff log and status updates are marshalled through the app dispatcher, including the reset-time return from Expeditions to Challenges.
- File logging remains immediate and does not depend on the window still accepting UI updates.

## Setup guide

- Added a persistent **Setup guide** button above **Join Discord** in the left sidebar.
- The button opens the public Google Docs setup walkthrough in the default browser.

## Validation

- Replayed the reported automatic diagnostic capture and confirmed the cooldown selector reaches the intended Expeditions handoff.
- Passed the full automated test suite and all checked-in detector fixtures.
- Rendered and reviewed the complete dark/light UI snapshot set.
- Existing camera models, placement models, Expeditions presets, Challenge presets, and detector pack 1.0.1 remain compatible.

## Downloads

- `ExpeditionsMacro-1.1.1-win-x64-setup.exe` - recommended Windows installer.
- `ExpeditionsMacro-1.1.1-win-x64.zip` - portable self-contained build.
- `anime-expeditions-expeditions-1.0.1.zip` - compiled detector pack.
- `SHA256SUMS.txt` - SHA-256 checksums for every release asset.

Windows 10 or Windows 11 x64 is required. The binaries are currently unsigned, so Windows may identify the publisher as unknown. Verify the checksums before running a downloaded artifact.
