# Expeditions Macro v1.1.3

## Highlights

- Roblox is resized to the standard 808 by 611 client size when the app opens if Roblox is already running.
- Automation now leaves Roblox at the standardized client size after setup, captures, placements, and macro runs. This avoids later geometry drift on machines where Roblox changes its minimum window size.
- Diagnostic ZIPs can include the current macro run log, controlled by a new Settings toggle.
- Challenge cooldown fallback now finishes the active Expedition run before returning to Challenge mode when a global reset arrives.
- The fallback closes the Expedition result screen before changing modes, preventing the reported `Timed out waiting for PostmatchPreview` stop.

## Notes

Review diagnostic ZIPs before sharing when log inclusion is enabled. The app does not upload them automatically, but logs can include local paths and run details.
