# Expeditions Macro v1.3.0-beta.15

This silent beta adds an isolated Debug workspace, makes camera setup denser and faster, avoids redundant preparation on verified repeats, and hardens animated navigation controls. It does not send a Discord release announcement.

## Debug workspace

- Enable **Debug workspace** in Settings to reveal production-backed Play navigation and saved-team test tools.
- Navigation tests support explicit lobby and post-match starts, the selected mode and preset, and Challenge type selection. They stop at verified prestart before camera alignment, team loading, placement, or Start Game.
- Team tests begin and end with Units closed and use the production Team 1 through 8 scrollbar and Load Team verification.
- Run continuously, pause before inputs, or pause after each detector observation. Previous and Next review captured history, Step authorizes one pending live boundary, and Fast-forward resumes without further gates.
- When Deep Debug is enabled, archives include the selected Debug tool, preset or team, step mode, checkpoints, frames, detections, and resulting input events.

## Camera and repeated runs

- New schema 4 camera models capture a signed fine-yaw neighborhood, then sample a dense full turn at up to roughly 60 FPS while Right Arrow remains held.
- Setup uses independent visual evidence to recognize the completed loop, re-observes the stationary pose after key release, and returns through bounded closed-loop corrections.
- Runtime can use a strong, isolated dense fingerprint for a short closed-loop correction, while fine-yaw and three-frame full-resolution goal verification remain mandatory.
- Verified same-route repeats keep the already-loaded team and aligned camera. Navigation, recovery, interruption, or route changes safely invalidate the reusable preparation.

## Navigation, reporting, and packaging

- Detector-owned buttons must stop moving before input, preventing Play, Challenge, Expedition, Story, Raid, and Unit Teams opening animations from supplying stale click coordinates.
- Unit Teams waits for the real scrollbar thumb to settle and restores the expected top position before aligning the requested team.
- Discord totals now preserve whole-plan runtime, victories, and defeats across mode changes while retaining per-match runtime.
- Portable ZIPs expose `ExpeditionsMacro.exe` directly at the archive root.
- Switch knobs retain their complete circular shape in enabled, disabled, and focused states at supported Windows scaling.

## Validation

- All 619 application/detector tests and all 5 Deep Debug Viewer tests pass.
- Repository policy, release metadata, the warning-free Release build, all 24 dark/light UI snapshots, packaged portable startup, installer startup/uninstall, and release-asset verification pass.
