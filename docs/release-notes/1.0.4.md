# Expeditions Macro 1.0.4

This patch makes camera-model setup recognize a genuine full yaw turn when its return frame is slightly degraded, then measures the exact turn with fine mouse drags.

## Changes

- A degraded return peak can now close the yaw scan when the following view also matches the beginning of the learned sequence.
- An isolated lookalike landmark elsewhere in the turn is rejected unless its following yaw view confirms the same sequence.
- After finding the wraparound neighborhood, setup sweeps both directions with the configured fine-drag size and uses stable multi-frame scores to select the best position.
- Camera models store the refined full-turn circumference instead of rounding it to a coarse-drag multiple, and their terminal atlas frame is replaced with the refined goal view.

Create or recreate a camera model with this version to record the refined circumference. Existing models remain usable.
