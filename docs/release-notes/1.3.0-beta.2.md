# Expeditions Macro v1.3.0-beta.2

This silent beta fixes three automation boundaries reported against beta.1 and shortens the wait for testable prerelease artifacts. It does not send a Discord release announcement.

## Reliability fixes

- Story now clicks the stable map-copy area of its Play tile instead of the reward-icon strip, where a gem tooltip could absorb the click.
- Story and Raid recognize the current three-button party preview and click its detected Start button. The new matcher is scoped to that exact layout so ordinary Expedition screens are not mistaken for a preview.
- Camera preparation enables shift lock before the first pitch or fine-yaw drag and keeps it enabled for every mouse drag. Cleanup disables it after success, cancellation, or failure, preventing the visible cursor from crossing the unit hotbar.

## Faster beta feedback

- Fast tests, the complete golden-image corpus, and dark/light UI snapshots now run as three parallel GitHub Actions jobs.
- Silent beta packaging starts at the same time and does not wait for validation, allowing testers to download the build sooner.
- Validation failures remain visible on the tagged commit and still block promotion to a stable release.

## Validation

- Passed all 382 local Release tests, including the complete cross-state image corpus.
- Added a reviewed 808 by 611 Raid party-preview fixture with its account strip redacted before publication.
