# Expeditions Macro 1.0.8

This patch fixes a long-running pause stall caused by Roblox retaining a button's hovered appearance after the macro moved the Windows cursor away.

## Changes

- Added a persistent **Join Discord** button in the app sidebar that opens the public community invite in the default browser.
- Every absolute cursor move used for clicking or parking is now followed by a `+1/-1` relative motion pulse. This makes Roblox acknowledge the pointer's actual location instead of retaining its previous internal hover position.
- After a click, the cursor is parked at a neutral client edge and the macro waits long enough for Roblox to render the non-hovered control before visual detection resumes.
- The fix applies through the shared click path, covering Start, Continue, checkpoint, confirmation, reward, lobby, recovery, victory/defeat, placement, and other automated clicks.

Existing camera models, placement models, presets, and detector pack 1.0.1 remain compatible.
