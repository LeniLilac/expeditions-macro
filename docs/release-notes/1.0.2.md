# Expeditions Macro 1.0.2

This patch makes difficulty selection resilient to the latest game UI position change and finishes several interface cleanup fixes.

## Changes

- Difficulty verification now reads the active control's distinct green, red, or magenta color instead of depending only on a grayscale template at one exact Y coordinate.
- The detector accepts both the original layout and the control's new six-pixel-up position.
- Difficulty animations are followed as visible state changes, and one target match is still enough to continue.
- Preset, camera-model, and placement-model selectors now show only their saved names.
- Placement status text and Expeditions status content no longer overlap action controls at constrained widths.
- Three captures from the shifted layout are included as public regression fixtures.

## Downloads

- `ExpeditionsMacro-1.0.2-win-x64-setup.exe` — recommended Windows installer.
- `ExpeditionsMacro-1.0.2-win-x64.zip` — portable self-contained build.

Windows 10 or Windows 11 x64 is required. The binaries are currently unsigned, so Windows may identify the publisher as unknown. Verify the checksums before running a downloaded artifact.
