# Expeditions Macro v1.1.4

## Highlights

- Existing camera and placement models can now be renamed without changing their saved model IDs or breaking presets that already reference them.
- Expeditions and Challenges no longer show detector-pack selectors. The app uses the installed current detector pack automatically, while detector-pack details and updates stay in Settings.
- Expeditions now scrolls properly when the window is shorter or tuning fields are expanded.
- The Expeditions and Challenges configuration areas have been tightened into clearer route, model, recovery, and reporting groups.
- Page title descriptions were removed from Expeditions, Challenges, Camera Models, Placement Models, and Settings to reduce repeated copy.
- New installs default diagnostic log inclusion to off. Users must explicitly enable both automatic error captures and log inclusion.
- Challenge cooldown fallback can now return from an already-open Expedition party preview, so it does not get stuck clicking the bottom-left Play button when **Change Gamemode** is already visible.

## Notes

Existing models, presets, challenge presets, and detector pack 1.0.1 remain compatible.
