# Expeditions Macro v1.1.5

## Highlights

- Fixed Challenge mode returning from its cooldown Expeditions run when Roblox is already showing the compact three-button Expedition party preview.
- The macro now recognizes that specific panel structure and clicks the detected **Change Gamemode** button at its live position instead of retrying the bottom-left Play coordinate.
- Compact preview recognition requires the adjacent purple **Change Map** and yellow **Change Gamemode** controls, preventing unrelated narrow yellow UI elements from triggering a mode switch.
- Added the reported v1.1.4 screen as a privacy-redacted golden regression fixture and reran the full cross-state detector corpus.

## Notes

Existing models, presets, Challenge presets, and detector pack 1.0.1 remain compatible.
