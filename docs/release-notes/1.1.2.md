# Expeditions Macro 1.1.2

This is the first public release of the new Challenge mode. It automates the complete regular-Challenge rotation and can continue farming Expeditions between Challenge resets.

## Highlights

- Added a dedicated **Challenges** workspace for the Trait, Stat, and Sprite Challenges that reset globally at `xx:00` and `xx:30`.
- Detects School Grounds, Flower Forest, Rose Kingdom, Fairy King Forest, and King's Tomb, then loads the matching camera and placement models automatically.
- Supports separate before-start and delayed after-start placements, configurable defeat retries, per-reset attempt tracking, and daily-limit waiting.
- Can run a configured Expeditions preset while all selected Challenges are on cooldown, then return when the next Challenge window opens.
- Adds optional Components V2 Discord reports, five restricted error pings to a configured user, and automatic diagnostic screenshot ZIPs after unexpected failures.

## Reliability

- Challenge mode now closes the active or gray cooldown selector and requires the game-mode selector to be recognized before it launches the configured Expeditions preset.
- The verified game-mode selector is handed to the existing Expeditions recovery route, which opens Expeditions, selects the configured map and difficulty, starts the private stage, and waits for prestart.
- If Roblox ignores the selector close click, the macro retries the focused action up to three times instead of waiting indefinitely for an unreachable Expedition prestart screen.
- A persistent selector now produces an actionable error and the configured automatic diagnostics rather than a silent stall.
- Includes the post-teleport recovery, confirmation-dismissal, registered-click, and detector hardening accumulated through the Expeditions releases.

## Setup

- Added a persistent **Setup guide** button in the sidebar that opens the public walkthrough in the default browser.
- Existing camera models, placement models, Expeditions presets, Challenge presets, and detector pack 1.0.1 remain compatible.

## Validation

- Replayed all 149 frames of the supplied manual selector-to-Expeditions route.
- Added privacy-safe cooldown-selector and game-mode-selector frames to the public regression dataset.
- Passed 232 automated tests across 339 checked-in Roblox client captures.
